〜サイトのイメージ〜

コマンドで操作しているようなCUIベースのホームページ

例
　show profile　→　画面上にプロフィール情報を出力する。（枠はブラウザチックなデザイン。）
　　　　　　　　　　＋α　よく映画などで映るFBIの犯罪者リストみたいな感じにしたい。

　show blog　 　→　技術ブログ的なページを表示

　show info	→　連絡先の表示

　mode GUI	→　GUIベースのレイアウトに変更
　　　 CUI	→　CUIベースのレイアウトに変更


基本的な画面の流れ
　１．ページの上の入力フォームにコマンドを入力して Enter
　２．ページ読み込みのプログレスバー表示
　３．GUIのページを画面上に表示
　　　＋α　ページの左上に閉じるボタンを設定して、それを押したらページを非活性化
　　　＋α　ページの開く閉じる際は、アニメーションを使いたい。。

懸念点
　レスポンシブが難しい。
　　→　スマホ等のタブレット機器の場合、常にGUIモードでページを表示する。
　　　

　何度もコマンドを入力するのが鬱陶しい可能性がある。
　　→　GUIモードに変更した場合は、Cockieにデータを保存して
　　　　ページを離れても、付き訪れるときはGUIモードで開くように工夫する。。（理想）

コマンドを入力後エンターキーを押下することで画面を表示する。
　　→　モーダルを使う？
　　　　→　登録、API通信などがある場合、実装が複雑になりそう。厳しいかも。。。
